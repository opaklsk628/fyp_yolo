Telegram Notification Settings

How to get Telegram Bot Token and Chat ID:

  1. Open Telegram and search for @BotFather
  2. Send /newbot and follow instructions to create a bot
  3. Copy the bot token (looks like: 123456789:ABCdefGHIjklMNOpqrsTUVwxyz)
  4. Start a chat with your bot
  5. Send a message to your bot
  6. Visit: https://api.telegram.org/bot[YOUR_BOT_TOKEN]/getUpdates
  7. Find your chat ID in the response (usually a number like 123456789)
